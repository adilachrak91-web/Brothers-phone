# Brothers-Phone-Achrak-A | نسخة الويب 2026

نظام إدارة متجر الهواتف **Brothers-Phone** ونسخة ويب مجانية وكاملة للبرنامج الأصلي، تم تطويرها باستخدام **HTML5 + CSS3 + JavaScript** بدون الحاجة لـ Node.js.

## ✨ المميزات

### 🛒 نقطة البيع (POS)
- بحث سريع عن المنتجات بالباركود أو الاسم
- إضافة منتجات للسلة بسهولة
- تطبيق خصومات وزيادات على الأسعار
- تأكيد المبيعات وإدارة المرتجعات
- معالجة المنتجات التالفة

### 📦 إدارة المستودع (Inventory)
- إضافة وتعديل وحذف المنتجات
- تتبع الكميات والأسعار
- حساب رأس المال التلقائي
- البحث السريع عن المنتجات
- تحديد السلع الناقصة (أقل من 3 وحدات)
- رفع صور للمنتجات

### 📊 التقارير والأرشيف
- عرض تفصيلي للمبيعات اليومية
- حساب الأرباح والخسائر
- إدارة المصروفات (شخصية واستثمارية)
- تتبع الافتتاحية والإقفالية
- إحصائيات يومية شاملة

### 💸 الديون والكريديت
- إدارة ديون العملاء
- إدارة ديون الموردين
- تسجيل التسديدات الجزئية
- حساب المتبقي من الديون
- حذف الديون المسددة

## 🚀 البدء السريع

### طريقة 1: فتح الملف مباشرة
1. انقر بزر الماوس الأيمن على ملف `index.html`
2. اختر "فتح مع" ثم اختر متصفح ويب (Chrome, Firefox, Edge, إلخ)
3. التطبيق سيعمل فوراً!

### طريقة 2: استخدام خادم محلي (موصى به)

#### على Windows (PowerShell):
```powershell
# تشغيل خادم Python المدمج
python -m http.server 8000

# أو استخدام
python -m SimpleHTTPServer 8000

# ثم افتح المتصفح على:
# http://localhost:8000
```

#### على Mac/Linux:
```bash
# تشغيل خادم Python المدمج
python3 -m http.server 8000

# ثم افتح المتصفح على:
# http://localhost:8000
```

## 💾 حفظ البيانات

- جميع البيانات تُحفظ **محلياً** في متصفحك باستخدام **localStorage**
- لا يتم إرسال أي بيانات إلى الإنترنت
- يتم النسخ الاحتياطي التلقائي للبيانات
- يمكنك استخراج البيانات في أي وقت

## 🎯 دليل الاستخدام

### إضافة منتج جديد
1. اذهب إلى تبويب **📦 إدارة المستودع**
2. املأ بيانات المنتج (باركود، اسم، أسعار، كمية)
3. اختياري: رفع صورة للمنتج
4. اضغط **💾 حفظ**

### بيع منتج
1. اذهب إلى تبويب **🛒 نقطة البيع**
2. ابحث عن المنتج (باركود أو اسم)
3. اضغط Enter أو على زر البحث
4. المنتج سيضاف للسلة
5. اضغط **✅ تأكيد البيع** عندما تنتهي

### إدارة المرتجعات
1. في تبويب **🛒 نقطة البيع**
2. استخدم قسم **🔄 مرتجع زبون**
3. أدخل الباركود والمبلغ
4. اضغط **إرجاع**

### عرض التقارير
1. اذهب إلى تبويب **📊 التقارير**
2. اختر التاريخ المطلوب
3. ستظهر جميع المبيعات والمصروفات
4. يمكنك إضافة مصروفات جديدة

### إدارة الديون
1. اذهب إلى تبويب **💸 الديون والكريديت**
2. أضف دين جديد (زبون أو مورد)
3. استخدم زر **تسديد** لتسجيل الدفعات
4. حذف الديون المسددة

## ⚙️ المتطلبات

- ✅ متصفح ويب حديث (Chrome, Firefox, Edge, Safari)
- ✅ تفعيل JavaScript
- ✅ لا تحتاج Node.js أو npm
- ✅ لا تحتاج توصيل بالإنترنت

## 🔒 الأمان والخصوصية

- جميع البيانات تُخزن **محلياً على جهازك**
- لا يتم تحميل البيانات إلى السيرفر
- يمكنك حذف البيانات في أي وقت من إعدادات المتصفح
- تطبيق آمن تماماً للاستخدام في المتجر

## 🛠️ تطوير التطبيق

إذا أردت تطويراً متقدماً (React, Database, إلخ):

1. **نسخة React**: ستحتاج Node.js و npm
2. **نسخة الويب الكاملة**: يمكن إضافة Node.js backend وقاعدة بيانات
