# 🚀 دليل البدء السريع

## خطوات التشغيل البسيطة

### الطريقة الأولى: فتح الملف مباشرة في المتصفح ✅ (الأسهل)

1. **انقر مزدوجاً على ملف `index.html`**
   - أو انقر بزر الماوس الأيمن → اختر "فتح مع" → اختر متصفحك المفضل

2. **التطبيق سيحمل فوراً!** 🎉

**ملاحظة:** هذه الطريقة تعمل بنسبة 100% في جميع الحالات

---

### الطريقة الثانية: تشغيل خادم Python (موصى به للأداء الأفضل)

#### Windows:

```powershell
# افتح terminial PowerShell في المجلد
# ثم اكتب:

python server.py
```

أو بطريقة يدوية:

```powershell
python -m http.server 8000
```

ثم افتح المتصفح على: **http://localhost:8000**

#### Mac/Linux:

```bash
python3 server.py
```

أو:

```bash
python3 -m http.server 8000
```

ثم افتح المتصفح على: **http://localhost:8000**

## 🆘 حل المشاكل

### المشكلة: الصفحة بيضاء
**الحل:** 
- جرب متصفح مختلف
- امسح ذاكرة التخزين المؤقتة (Ctrl+Shift+Delete)
- جرب تشغيل الخادم بدلاً من الفتح المباشر

### المشكلة: لا تظهر البيانات
**الحل:**
- تأكد من تفعيل JavaScript في المتصفح
- جرب متصفح آخر

### المشكلة: بطء التحميل
**الحل:**
- استخدم الخادم Python (طريقة أسرع)
- امسح الذاكرة المؤقتة
